Tutorial para rodar a aplicação:
1) criar na raiz do projeto 2 pastas chamadas: uploads e downloads
2) baixar o modelo de transcrição small.pt no seguinte link https://openaipublic.azureedge.net/main/whisper/models/small.pt  colocar esse arquivo dentro de uma pasta com o nome models_whisper
3) renomear o arquivo .nv.exampla para .env e colocar as informações do banco de dados
4) para rodar local utilizei o xampp para emular o banco de dados no phpmyadmin https://www.apachefriends.org/pt_br/download.html
